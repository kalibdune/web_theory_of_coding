Проверка на чётность
Основная теория
Проверка четности – очень простой метод для обнаружения ошибок в передаваемом пакете данных. С помощью данного кода мы не можем восстановить данные, но можем обнаружить только лишь одиночную ошибку. Данный алгоритм используется в составе других сложных помехоустйочивых кодах.
В каждом сообщении есть один бит четности. Этот проверочный бит устанавливается во время отправки данных, после чего рассчитывается и сравнивается во время получения данных. Он равен сумме по модулю 2 всех бит данных в пакете. То есть число единиц в пакете всегда будет четно. Изменение одного бита в любом месте кода сообщает о возникшей ошибке.
