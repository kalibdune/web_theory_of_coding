Цель проекта
Наш проект приследует цель: помочь школьникам разобраться в помехоустойчивом кодировании.
Мы предоставляем ученикам пособие по кодам: проверка на чётность, код Хэмминга и код Рида Соломона.
Также нашей командой был разработан генератор задач для интелектуального мегаполиса в качестве отработки пройденного материала по коду Хэмминга.
демонстрация кодов онлайн

Десктопное приложение

В рамках школьного конкурса проектных работ, 2021-2022 годов, нашей командой было разработано open source десктопное приложение для наглядной демонстрации помехоустойчивых кодов.
приложение работает на платформах: Windows 7/10/11 и дистрибутивах linux.
